# Renovation Manager

This is a personal project designed to help manage DIY (Do-It-Yourself) projects. Its main features include: 
- providing an overview of all ongoing and planned DIY projects
- tracking budget
- defining the list of necessary materials
- outlining the steps required for each project 
This project aims to make organizing and completing DIY tasks easier and more efficient.


## Tech stack

It is based on [Symfony-docker template] (https://github.com/dunglas/symfony-docker):

A [Docker](https://www.docker.com/)-based installer and runtime for the [Symfony](https://symfony.com) web framework,
with [FrankenPHP](https://frankenphp.dev) and [Caddy](https://caddyserver.com/) inside!
